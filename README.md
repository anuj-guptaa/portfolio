# README
# Professional Portfolio Website

A clean, professional, and fully responsive portfolio website built with React, Vite, Tailwind CSS, and Docker. Perfect for showcasing your coding projects to recruiters and potential employers.

## Features

- ✨ **Modern Design**: Clean, professional interface with smooth animations
- 📱 **Fully Responsive**: Works perfectly on all devices (mobile, tablet, desktop)
- 🚀 **Fast Performance**: Built with Vite for optimal load times
- 🎨 **Beautiful UI**: Tailored with Tailwind CSS for a sleek appearance
- 🐳 **Dockerized**: Single service, easy deployment with Docker
- ♿ **Accessible**: Built with accessibility best practices
- 📧 **Contact Form**: Functional contact form for inquiries
- 🔗 **Social Links**: Easy navigation to your GitHub, LinkedIn, and other profiles

## Getting Started

### Prerequisites

- Node.js 18+ (for local development)
- Docker & Docker Compose (for containerized deployment)

### Local Development

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```
   Visit `http://localhost:5173` in your browser

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Preview production build**:
   ```bash
   npm run preview
   ```

### Docker Deployment

1. **Build and run with Docker Compose** (recommended):
   ```bash
   docker-compose up --build
   ```
   This uses the **development** stage with hot reload enabled. Changes to your code will automatically refresh in the browser.
   
   Visit `http://localhost:5173` in your browser

2. **Production build with Docker**:
   ```bash
   docker-compose -f docker-compose.prod.yml up --build
   ```
   This uses the optimized production stage with a static server.

3. **Or build and run manually**:
   ```bash
   # Build the Docker image
   docker build -t portfolio .

   # Run the container for development
   docker run -p 5173:5173 -v $(pwd):/app -v /app/node_modules portfolio

   # Or for production
   docker run -p 5173:5173 portfolio:production
   ```

## Customization

To make this portfolio your own:

1. **Update Personal Information**:
   - Edit `src/components/Hero.jsx` - Change title and description
   - Edit `src/components/Contact.jsx` - Add your email, location, social links
   - Edit `src/components/Footer.jsx` - Update name and year

2. **Add Your Projects**:
   - Open `src/components/Projects.jsx`
   - Update the `projects` array with your own projects
   - Add project details: title, description, tags, links

3. **Customize Skills**:
   - Edit `src/components/Skills.jsx`
   - Update `skillCategories` with your technologies

4. **Styling & Colors**:
   - Main colors are defined in `tailwind.config.js`
   - Modify the `colors` object to match your preferred palette
   - Primary blue: `#3b82f6` can be changed in the theme

5. **Form Handling**:
   - The contact form logs to console by default
   - To add email functionality, integrate with services like:
     - Formspree
     - EmailJS
     - Your own backend API
